# speed-mern-app-demo-backend

Speed your MERN app by REDIS

## Config

Please add "redis" and "mongodb" server connection string in the .env file

```js
//.env file
PORT=3000
MONGODB_URL=
REDIS_URL=
```

## Run project locally

```sh
npm install

npm start
```
